# SmartResto — Автоматизована система ресторанного обслуговування

Екосистема для замовлень через QR-коди, KDS (кухонних екранів) та сервісних викликів.

## Стек технологій
- **Frontend:** Next.js (TypeScript, Tailwind CSS)
- **Backend:** NestJS (TypeScript, Prisma ORM)
- **Бази даних:** PostgreSQL, Redis
